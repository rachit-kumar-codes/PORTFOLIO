## 🔍 Overview

This portfolio is built as a single continuous scroll — Hero → About →
Skills → Projects → Footer — rather than a multi-page app, with every
section designed around a consistent dark, glow-accented, `< />` terminal
visual language.

## ✨ Highlights

- **Single-page, anchor-based navigation** — no route changes, just smooth
  scrolling between sections.
- **Interactive skills showcase** — hover a skill (or press its matching
  key on your keyboard) to reveal its name and description. It's a normal
  in-flow section, not a fixed background layer, and only "activates" while
  actually in view.
- **Animated starfield background** — a lightweight canvas-based particle
  field that reacts subtly to the cursor, fixed behind the entire page.
- **Terminal-styled footer** — `whoami` / `projects` prompts, a blinking
  cursor, and glowing social icons.
- **Glowing, animated Education card** — an isolated, reusable component
  with a rotating gradient border and pulse glow.
- **Themed project cards** — trimmed, badge-based tech-stack tags instead
  of plain comma-separated text, with GitHub/Live Demo actions.
- **Fully responsive** — a proper mobile-first breakpoint system (360px →
  1200px+), not just a single "mobile" cutoff.
- **Plain JavaScript** — no TypeScript, no unnecessary build complexity.

## 🚀 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/rachit-kumar-codes/portfolio.git
cd portfolio
npm install
```

Run the dev server:

```bash
npm run dev
```

Lint the project:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```

Output is written to `dist/`.
